# Phase 02.1 Plan 01: Refatoração SkinConfig para ApplicationV2 e Estilos Dinâmicos

## Objective

- **What:** Refatorar a interface `SkinConfig` para os novos padrões do Foundry V14 (ApplicationV2, Actions, `<file-picker>`, `<color-picker>`) e migrar a aplicação de configurações de fonte para atualização dinâmica de CSS (sem reload).
- **Why:** Garantir compatibilidade nativa com Foundry V14, remover dependências legadas e melhorar a experiência do usuário com atualizações de estilo em tempo real.
- **Output:** Códigos refatorados em `src/scripts/apps/skin-config.ts`, `static/templates/skin-config.hbs` e `src/scripts/main.ts`.
